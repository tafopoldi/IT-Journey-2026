# Day 01 - Cloud Fundamentals

## What is Cloud Computing?

Cloud Computing means using computing resources such as servers, storage and networking over the internet instead of owning physical hardware.

---

## What is Microsoft Azure?

Microsoft Azure is a cloud platform provided by Microsoft that allows companies to rent IT resources such as virtual machines, storage, databases and networking services.

---

## Service Models

### IaaS (Infrastructure as a Service)

Microsoft manages:
- Hardware
- Networking
- Storage

Customer manages:
- Operating System
- Applications
- Data

Example:
- Azure Virtual Machine

---

### PaaS (Platform as a Service)

Microsoft manages:
- Hardware
- Networking
- Operating System

Customer manages:
- Application
- Data

Example:
- Azure App Service

---

### SaaS (Software as a Service)

Microsoft manages:
- Everything

Customer:
- Uses the software

Examples:
- Microsoft 365
- Outlook
- Teams

---

## Benefits of Cloud Computing

### Scalability
Resources can grow when demand increases.

### Elasticity
Resources can automatically increase or decrease as needed.

### High Availability
Services remain available even if part of the infrastructure fails.

### Reliability
Systems operate consistently and predictably.

### Security
Cloud providers invest heavily in protecting systems and data.
